# Placeholder para ícono 192x192

Para generar los íconos de la app:

1. Crear un ícono 512x512 con:
   - Fondo verde esmeralda (#10b981)
   - Símbolo de picas (♠) o cartas en blanco
   - Diseño minimalista

2. Usar una herramienta como:
   - [Favicon Generator](https://favicon.io/)
   - [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
   - Photoshop / Figma / Canva

3. Generar dos tamaños:
   - icon-192.png (192x192px)
   - icon-512.png (512x512px)

4. Reemplazar este archivo markdown con la imagen real.

Por ahora, el manifest.json está configurado pero los íconos son opcionales.
La app funcionará sin ellos, solo que no tendrá un ícono personalizado al instalarla.
